# Next.js training

[https://nextjs-training.fermercadal.vercel.app/](https://nextjs-training.fermercadal.vercel.app/)

Basic blog website built using Next.js.
* Navigate between pages
* Assets, metadata, CSS
* Pre-rendering, data fetching
* Dynamic routes
* API routes
* Deploy

## Installation

```bash
yarn install
```

## Usage

```bash
yarn run dev
```
